Running Workout Generator


Welcome to the information webpage for the Running Workout Generator. I used my running knowledge from cross country to try and create a mobile app that generates the perfect user specific running workouts. It is a unique app as there is not much info on creating good distance running workouts online and certainly no apps doing it for you. My goal for the app is for it to be useful for newer distance runners. It is soon to be available on the App Store!

Running Nerd Vocabulary

Slow Jog

 Barely faster than a walk

Faster Jog

 Almost a run

Easy Run Pace

 A slow running pace that is very comfortable. You should be able to sustain a casual conversation.

Steady Pace

 2-2:30 min/mile slower than 5k pace. Not hard but not easy.

__% Max Pace

 whatever % of your max effort on a given distance. eg 60% of your max effort for 800m

Tempo

 20 seconds slower than 5k pace. Should be able to hold for 30 minutes.

Near Max Effort

 Almost all out

Max Effort

 All out! 105%!
